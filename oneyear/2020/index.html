# 我的2020

# 题外话

时间倒回到 2020 年 1 月，在外婆家待了数周。微博被新型传染病刷屏时，各电视台还正放着即将过年的喜庆画面，几天后，CCTV13 终于报道了这一新闻，那时人们还不知道几天后将发生什么。

鄂A、奔驰、红十字会等各种事件让我对这个社会重新思考，一场疫情照出了无数妖魔鬼怪，因为政治敏感被封了两个账号，庆幸的是我依然对未来充满希望。

# 学习

期末开始着手寻找实习，起因是辅导员在群里发了 CVTE 的寒假实习招聘公告，正巧在广州，薪资也很诱人，二话不说投递简历。头铁的我顺手投了字节，花了一周时间准备面试，结果不尽如人意，没刷过几道 leetcode 的我倒在了算法上。两天后的 CVTE 我也毫不意外地挂了，我那时才反应过来自己没有准备笔试...输出结果这类题对我来说一头雾水。

“有人寒假想来北京实习吗”，看着实验室学姐在群里发的消息，我毫不犹豫回了“举手”。之后便顺利地通过了一二面拿了小米 offer ，这两年我总是感觉自己一事无成，看着 offer 的我开心地睡不着觉，这对我来说是个巨大的鼓舞，原来已经不知不觉领先于许多同学了。

在虎扑上发的一篇关于前端学习路径的文章已经有33W阅读量和579个推荐，这让我深刻体会到知识输出和帮助他人的乐趣，同时还有几位网友加我vx表示请教，我也给予了热心帮助。

家里人把考研看的太重，不太想让我去实习，理由自然是耽误时间，我觉得5月开始不算太晚，何况数学已经开始准备了（2月份留：想跨专业曲线救国了），实习经历一定能在复试取得不小的优势。决定去北京之后租一间不超过 3000 的单间，多花些时间在学习上~。

最近我独自承包了整个课程项目的开发工作，因为我知道等待接口再加上前后端联调到猴年马月也不一定能完成，自己做效率是最高的。所以最近学了 express 的很多东西，这比flask简单太多，而且前后端都是 js 思路也不需要太大改变，db.query可以直接用SQL让我甚是欣慰。同时希望能早日学习mongodb以及nodejs的进阶内容。

# 对2021和未来

希望自己编码能力有所提高，能拿一次奖学金，考研顺利。希望有生之年也能做一个 stars 上千的开源项目，那绝对是我的高光时刻。
