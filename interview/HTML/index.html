# meta标签

元数据（Metadata）是数据的数据信息。

<meta>标签提供了 HTML 文档的元数据。元数据不会显示在客户端，但是会被浏览器解析。

<meta>标签永远位于 head 元素内部。

<meta name="keywords" content="HTML,ASP,PHP,SQL">

META元素通常用于指定网页的描述，关键词，文件的最后修改时间，作者及其他元数据。

元数据可以被使用浏览器（如何显示内容或重新加载页面），搜索引擎（关键词），或其他 Web 服务调用。

# 语义化

怎么知道自己的页面结构是否语义化，那就要看你的HTML结构，在去掉CSS样式表之后，是否，依然能很好的呈现内容的结构，代码结构。也就是说，脱掉css的外衣，依然头是头，脚是脚。赤裸裸的完整的一篇文档。这也就是，语义化之后文档的效果。

# <script>标签会阻塞加载吗

会 <scritp>标签每次出现，都会让页面等待脚本的解析和执行，无论当前的 JavaScript 代码是内嵌的还是包含在外链文件中，页面的下载和渲染都必须停下来等待脚本执行完成。
